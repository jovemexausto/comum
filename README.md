# Comum Protocol

Infraestrutura de confianca minima para coordenacao soberana, offline-first
e sem autoridade central. Este repositorio segue disciplina de projeto
no estilo IETF, com especificacao via CIPs, vetores de teste e referencias
de implementacao.

Estrutura principal:

- spec/cips: CIPs normativos
- spec/registries: registros oficiais (verbos, suites, envelopes)
- spec/schemas: CDDL e esquemas CBOR
- spec/test-vectors: vetores de teste iniciais
- impl: implementacoes de referencia
- tests/conformance: conjunto de conformidade

Documentos-chave:

- spec/cips/CIP-0001.md
- GOVERNANCE.md
- SECURITY.md
