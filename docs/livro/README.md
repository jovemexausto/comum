# Esboco do livro do Comum

Objetivo: registrar a arquitetura inicial do futuro livro/whitepaper teorico,
filosofico e politico do Comum, mantendo-o separado da documentacao tecnica do
repo.

## Natureza deste corpus

Este livro:

- nao e especificacao
- nao e manual de implementacao
- nao e changelog do projeto
- nao e contrato tecnico

Ele existe para elaborar o campo teorico e politico do Comum com profundidade,
usando engenharia apenas o suficiente para sustentar os argumentos.

## Perguntas centrais

- como coordenar sem centro?
- como formalizar sem colonizar?
- como interoperar sem reduzir a pluralidade a uma equivalencia unica?
- como manter soberania local sem cair em isolamento?
- como ter instituicoes forkaveis sem reduzir a vida a codigo?
