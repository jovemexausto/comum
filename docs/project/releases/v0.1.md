# Release Notes v0.1

Data: 2026-04-06
Status: release

## Resumo

v0.1 estabelece a base normativa do Comum Protocol e entrega as
implementacoes de referencia em Rust e JS, runtime WASM, testes de
conformidade e vetores oficiais.

## Destaques

- CIP-0001 completo com registries, CDDL e regras de canonicalizacao
- comum-rs com CBOR canonical, sync, CTE, crypto e runtime WASM
- comum-js alinhado (encode/verify, contextos, genesis)
- Provas de Contexto completas (proximity, beacon, place, vouch)
- Genesis com FROST e capsula Agora compilavel
- Conformance runner e vetores oficiais
- Interop matrix publicada

## Compatibilidade

- Primeira release de protocolo; sem migracoes anteriores

## Testes executados

- `cargo test -p comum-rs`
- `npm test` em `packages/comum-js`

## Notas

- Tag local criada: `v0.1`
