# Release Notes v0.2

Data: 2026-04-06
Status: prerelease

## Resumo

v0.2 consolida o Commoner facade, formaliza semantica da capsula Agora,
define perfis de transporte e entrega um wrapper JS hibrido com N-API
opcional. A base fica mais interoperavel e a simulacao passa a ser um
pacote separado.

## Destaques

- Commoner facade em `comum-rs` com validacao completa e fluxo de sync
- Semantica da capsula Agora com regras de votacao e testes
- Perfis de transporte normativos (BLE/NFC/QR/RNS)
- Wrapper JS hibrido (N-API opcional, fallback para CLI)
- Simulacao de Agora em pacote cargo dedicado
- README expandido com manifesto e guia de uso

## Compatibilidade

- Sem breaking changes de protocolo
- N-API e opcional; `comum-js` funciona com fallback

## Testes executados

- `cargo test -p comum-rs`
- `cargo check -p comum-napi`
- `cargo test -p capsula-agora`
- `npm test` em `runtime/js/comum-js`

## Notas de upgrade

- Para usar N-API: compilar `comum-napi` e definir `COMUM_NAPI_PATH`
- Simulacoes agora vivem em `apps/*/sims/` e `cases/*/sim/`
