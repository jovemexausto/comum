# PHI esperado por verbo (hipotese)

Objetivo: explicitar expectativas minimas de contexto (PHI) por verbo
para permitir analise de coerencia. Nao normativo.

## Feira (capsula-feira)

- offer:
  - PHI: nenhum obrigatorio

- accept:
  - PHI: deve referenciar offer
  - Opcional: encounter ou vouch

- receipt:
  - PHI: cadeia completa (offer + accept)

## Observacoes

- Estas sao expectativas locais (capsula), nao regras do protocolo.
- Servem apenas para analise e simulacao.
